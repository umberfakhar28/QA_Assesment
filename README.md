# WP Dark Mode Cypress Tests

## Installation

1. Clone the repo:
   ```bash
   git clone <https://github.com/umberfakhar28/QA_Assesment.git>
   cd UMBERFAKHAR
   ```

2. Install Cypress dependencies:
   ```bash
   npm install
   ```

3. Placed the scripts file in the `e2e` folder within the Cypress directory.
